# claude-os

A virtual AI OS you drive through Claude Code skills. Skills are the commands; a small kernel
(`bin/`, `systemd/`) handles hosting; your projects live as self-contained repos under
`services/`, `experiments/`, `packages/`, and `apps/`.

See [`CLAUDE.md`](./CLAUDE.md) for the model and conventions.

> Custom-first: currently tailored to one machine. Genericizing into a reusable open-source
> blueprint is planned.
