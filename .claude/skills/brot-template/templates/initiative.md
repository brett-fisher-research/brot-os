---
description: >
    A long-term initiative tracker — an AI+human goal spanning weeks/months (a product launch,
    learning piano), coarse and revisable where plans are decomposed and session-scoped. One file
    per initiative at .brot/initiatives/<slug>.md — gitignored, NEVER deleted. Human-readable
    first: a fresh session must re-orient fast from Status + Now / next + the latest log entry.
    NO checkboxes anywhere, NO bold emphasis.

    Status legend, used in the header and the Milestones table — always emoji + word, never
    emoji alone: 🟢 active/in motion · 💭 fuzzy · ⬜ not started · 🌙 paused · 🏁 done · 🧊 iced.

    Sections. Goal: the SMART end-state in 1-2 sentences — "done" must be unambiguous.
    Success measures: measurable signals the goal is met. Milestones: the coarse arc — fuzzy
    tail milestones are fine, mark them 💭 fuzzy. Now / next: the 1-2 things in motion — a fresh
    session reads this first. Log: newest first, dated ### headings with a one-line summary;
    body notes PRs merged, human work, feelings, pivots — author noted as "the user:" / "PM:"
    when it matters. Open questions: ambiguity parked for whiteboarding/research.

    Rendering: the code fence below delimits the banner art ONLY. The Milestones table (and all
    section prose) prints and is written to the file OUTSIDE any fence — a fenced table shows
    raw pipes.
